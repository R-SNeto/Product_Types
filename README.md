# 📦 Product Types

Projeto desenvolvido em **Java** com o objetivo de aplicar conceitos fundamentais de **Programação Orientada a Objetos (POO)**, com foco em herança, polimorfismo e classes abstratas.

O sistema simula o cadastro e a exibição de diferentes tipos de produtos, cada um com comportamentos específicos.

---
## 🎯 Objetivo

Demonstrar na prática como utilizar herança e polimorfismo para tratar diferentes tipos de produtos de forma unificada, incluindo:

- Produto comum
- Produto usado
- Produto importado

---
## 🏗️ Estrutura do Projeto

```text
Product_Types/
│
├── application/
│   └── Main.java
│   └── MainUI.java   
│
├── entities/
│   ├── Product.java
│   ├── UsedProduct.java
│   └── ImportedProduct.java
```
## 🚀 Funcionalidades
- Cadastro de produtos
- Diferenciação de tipos de produto (comum, usado e importado)
- Cálculo de preço total (incluindo taxas de importação)
- Exibição personalizada de informações de cada produto
- Manipulação de datas (LocalDate) para produtos usados
- Uso de listas (ArrayList)
  
---
## 🧠 Conceitos Aplicados
- Programação Orientada a Objetos (POO)
- Herança
- Polimorfismo
- Encapsulamento
- ENUMS
- Sobrescrita de métodos (override)
- Coleções (List)
- Manipulação de datas (LocalDate)

# 📦 Product Types

A project developed in **Java** focused on applying core **Object-Oriented Programming (OOP)** concepts, especially inheritance, polymorphism, and abstract classes.

The system simulates the registration and display of different product types, each with specific behaviors.

---

## 🎯 Purpose

To demonstrate how inheritance and polymorphism can be used to handle different types of products in a unified way, including:

- Common product  
- Used product  
- Imported product  

---

## 🏗️ Project Structure

```text
Product_Types/
│
├── application/
│   └── Main.java
│   └── MainUI.java   
│
├── entities/
│   ├── Product.java
│   ├── UsedProduct.java
│   └── ImportedProduct.java
|
├── entities_enums/
|   ├── ProductType.java
```

---
🚀 Features
- Product registration
- Different product types (common, used, imported)
- Total price calculation (including import fees)
- Customized product information display
- Date handling (LocalDate) for used products
- Use of collections (ArrayList)

---
🧠 Concepts Applied
- Object-Oriented Programming (OOP)
- Inheritance
- Polymorphism
- Encapsulation
- ENUMS
- Method overriding
- Collections (List)
- Date handling (LocalDate)
